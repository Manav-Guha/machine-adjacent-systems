# Machine Adjacent Systems

Corporate website for Machine Adjacent Systems (MAS), building verification frameworks for systems where claim and performance verification are mission-critical.

**Live:** [machine-adjacent-systems.vercel.app](https://machine-adjacent-systems.vercel.app)

## Products

- **myCode** — Runtime verification for AI-generated code. Corpus-grounded structural correspondence. Evidence-based declarative statements about code behaviour under specific conditions. [Try it.](https://mycode-ai.vercel.app)
- **MIAF** — Military Innovation Assessment Framework. Four-register analytical framework validated against 28+ historical cases.

## Stack

Static HTML, CSS, and JavaScript. No build step. Deployed via Vercel.

## Local development

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000` in a browser.

## Deployment

Connected to Vercel. Pushes to `main` deploy automatically.
